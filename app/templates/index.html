Exoplanet Detection

Machine Learning for Exoplanet Detection

We present two neural network models trained with the Exoplanet Hunting in Deep Space dataset from Kaggle.

The first model is based on an LSTM network; the data was normalized and a Gaussian filter was applied to smooth the signal. Finally, the dataset was balanced using SMOTE. The model achieved a sensitivity of 0.6 and a specificity of 0.99.

The second model is a multilayer perceptron trained in a similar way. In this case, the sensitivity and specificity were 0.4 and 0.94.

You can test the models with the Kaggle dataset or with our dataset.
